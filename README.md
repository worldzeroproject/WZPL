## Overview

The World Zero Project (WZP) is a unified platform consisting of multiple components, including ZScript, ZUserRuntime, ZServerRuntime, ZFramework, and the World Zero Project web application. Together, these systems provide deterministic logic execution, runtime orchestration, and declarative UI modeling.

This repository publishes and maintains the official WZPL license.

---

## Components

- ZScript – A deterministic scripting language for defining logic and processes
- ZUserRuntime – User-side runtime environment
- ZServerRuntime – Server-side runtime and execution host
- ZFramework – Shared abstractions and platform utilities
- World Zero Project Web Application – The official platform interface
